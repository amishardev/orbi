# **App Name**: Orbi

## Core Features:

- AI Community Finder: Suggest relevant communities to users based on their profile data and activity. This tool leverages AI to enhance community discovery by making a similarity comparison.
- Post Sharing: Users can share image posts with captions, similar to Instagram.
- Community Pages: Users can join or create community pages for colleges, clubs, and interest groups.
- Direct Messaging: Support for direct messaging between users and group chats within communities, akin to Discord.
- User Profiles: Users have profiles with info, pictures, bios, education, and joined communities.
- Community Join Requests: Users can request to join communities; admins approve/deny requests.
- Home Feed: Display posts from communities and connections, similar to Instagram's feed.

## Style Guidelines:

- Primary color: A vibrant blue (#4285F4) to convey trust and connectivity, aligning with the app's focus on networking and community.
- Background color: Light gray (#F5F5F5), providing a neutral backdrop that enhances readability and reduces visual fatigue.
- Accent color: A bright orange (#FF7043) for interactive elements, drawing user attention to calls-to-action and important features.
- Body text and Headings: Use 'PT Sans' (sans-serif), readable and modern, suitable for all text purposes.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean, modern icons to represent various actions and community types. Icons should be consistent and easily recognizable.
- Use a tab-based navigation for primary sections: Home Feed, Explore, Communities, Messages, and Profile. Ensure the layout is clean and intuitive for easy navigation.
- Subtle animations when loading new content or transitioning between pages. Keep animations minimal to avoid distracting users.
